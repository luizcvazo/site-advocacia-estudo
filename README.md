# Site de estudos - Dra. Lilia Rizatto Zaparolli

Este é um projeto de site institucional desenvolvido para a advogada **Dra. Lilia Rizatto Zaparolli**, com foco em apresentar suas principais áreas de atuação jurídica e facilitar o contato com clientes.

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias fundamentais do desenvolvimento web:

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização personalizada, layout responsivo (Flexbox e Grid) e efeitos visuais (Parallax/Zoom).
- **JavaScript (Vanilla)**: Lógica para o menu mobile, rolagem suave de links internos e manipulação do DOM.
- **Git & GitHub**: Versionamento de código e hospedagem via GitHub Pages.

## 📱 Funcionalidades

- [x] **Design Responsivo**: Adaptado para dispositivos móveis, tablets e desktops.
- [x] **Menu Mobile**: Navegação otimizada para telas pequenas.
- [x] **Interface Elegante**: Paleta de cores focada em profissionalismo (Preto e Dourado/Bege).
- [x] **Seção de Serviços**: Cards interativos detalhando as áreas do Direito Civil, Preventivo, Família e Previdenciário.
- [x] **Localização Integrada**: Google Maps incorporado para facilitar a visita ao escritório físico em Jaú - SP.
- [x] **Formulário de Contato**: Interface pronta para captação de leads.
